# stickers

The website for stickers.birki.io - Grant's collection of sticker drops!

## Rarity Aliases

For `public/data/*.json`, the `rarity` field supports shorthand aliases that map to the canonical
rarity strings used by the CSS effects:

- `holofoil` -> `Rare Holo`
- `common` -> `Common`
- `uncommon` -> `Uncommon`
- `galaxy` -> `Rare Holo Cosmos`
- `rare` -> `Amazing Rare`
- `radiant` -> `Radiant Rare`
- `holographic` -> `Trainer Gallery Rare Holo`
- `steel` -> `Rare Holo V`
- `ultra-rare` -> `Rare Ultra`
- `rare-rainbow` -> `Rare Rainbow`
- `holofoil-alt-1` -> `Rare Holo VMAX`
- `rare-rainbow-alt-1` -> `Rare Rainbow`
- `holofoil-alt-2` -> `Rare Holo VSTAR`
- `ancient` -> `Rare Secret`
- `shiny` -> `Rare Shiny`
